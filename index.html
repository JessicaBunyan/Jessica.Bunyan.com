Objects

Objects is a short game about identity, finding yourself, and also about Object Oriented Programming. Move around the playground and see which objects you can find, pick up variables and use them like items to solve problems. Objects is designed to convey the core prinicples of Objects Oriented Programming, without the player reading or writing single line of code.
