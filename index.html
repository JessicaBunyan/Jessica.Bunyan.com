Words of Ice and Fire

A textual analysis of the books behind HBO's Game of Thrones. Fans of the A Song of Ice and Fire series can use this web app to see the distribution & frequency of key words across each book and chapter.

This App is built with ReactJS and uses d3 and HTML5 canvas to graph the results. The dataset is generated from a C# backend which can be found on my Github

Objects

Objects is a short game about identity, finding yourself, and also about Object Oriented Programming. Move around the playground and see which objects you can find, pick up variables and use them like items to solve problems. Objects is designed to convey the core prinicples of Objects Oriented Programming, without the player reading or writing single line of code.
